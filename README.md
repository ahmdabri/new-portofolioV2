# Ahmad Ambari's Portfolio

A personal portfolio showcasing my projects, experiences, and skills as a Full-Stack Web Developer. 
Built with Next.js, Tailwind CSS, and GSAP for animations.

## Getting Started

First, install the dependencies and run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features
- Interactive UI and smooth animations with GSAP
- Responsive design using Tailwind CSS
- Fully optimized with Next.js App Router
